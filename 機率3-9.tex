By central limit theorem,\\
柯文哲的投票人數= X 分佈為：
\begin{align*}
&N(np, np(1- p)) = N(72, 36)\\
&Y = \frac{X - 72}{\sqrt[2]{36}} = N(0, 1) \\
&P(82 \leq X \leq 89) = P(\frac{(82-72)}{6} \leq Y \leq \frac{(89-72)}{6})\\
&= P(\frac{5}{3} \leq Y \leq \frac{17}{6}) = \phi(\frac{17}{6}) - \phi(\frac{5}{3})\\
&= 0.9977 - 0.9521
= 0.0456
\end{align*}
